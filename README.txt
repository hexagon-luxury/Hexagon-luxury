Hexagon Luxury - simple static website
Files included:
- index.html (homepage)
- style.css (styles)
- script.js (basic interactivity)
- products.json (example product data)
How to use:
1. Unzip the package.
2. Open index.html in your browser to view the site.
3. To deploy, upload the files to any static host (Netlify, Vercel, GitHub Pages) or your web server.
Fonts: Uses Montserrat from Google Fonts (online). If you need fully offline fonts, download Montserrat and adjust CSS.
Need changes? Reply with copy, product images, or branding assets and I'll update the site.
